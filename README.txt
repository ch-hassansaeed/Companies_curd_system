HTTP
----------------------------------------------------------------------------------
## Abstract
it use user_controller.js for companies curd client side operation and user_service.js for communication of Web-Service companies data through HTTP->REST API 
Main Page of Project is "CompaniesCURD.jsp" which is part of view.Model has company.java class which has companies structure,Controller is like broker which help to show web service data to Home page View page which is name as "CompaniesCURD.jsp",service name space is for define web service function and its implementation in UserServiceImpl.java it also has dummy companies data which will populate when site will preview.
